

# Greeter Smart Contract

A minimal Ethereum smart contract that stores and manages a greeting message.

## Description

This is a simple "Hello World" smart contract that demonstrates basic Solidity functionality:
- Stores a greeting string
- Allows reading the current greeting
- Allows updating the greeting (with event emission)
